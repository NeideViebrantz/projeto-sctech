# 🌐 Portfólio - Neide Viebrantz

Este projeto é uma **landing page de portfólio pessoal**, desenvolvido durante o curso **SCTech**, com o objetivo de apresentar meus projetos, habilidades e informações de contato como **desenvolvedora web iniciante**.

O site é responsivo, moderno e focado em **front-end** utilizando HTML, CSS e JavaScript.

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido como parte dos estudos no curso **SCTech**, com o objetivo de praticar:

* Estruturação de páginas com HTML
* Estilização com CSS
* Interatividade com JavaScript
* Organização de projetos para portfólio
  
---

## 📌 Funcionalidades

- Seção "Sobre Mim" com apresentação pessoal e trajetória profissional  
- Lista de projetos desenvolvidos durante o curso  
- Exibição das principais habilidades técnicas (HTML, CSS, JavaScript)  
- Seção de contato com e-mail, telefone, LinkedIn e GitHub  
- Layout moderno e responsivo  
- Botões e links interativos para navegar pelas seções  

---

## 🛠 Tecnologias utilizadas

- **HTML5** – Estrutura semântica das páginas  
- **CSS3** – Estilização, layout responsivo, animações básicas  
- **JavaScript** – Funcionalidades interativas  

---

## 🚀 Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/NeideViebrantz/projeto-sctech.git
```

2. Acesse a pasta do projeto:

```
cd projeto-sctech
```

3. Abra o arquivo **index.html** no navegador.
